## Wat er allemaal kan worden opgeslagen over een bestand:

- Originele bestandsnaam
- Unieke id
- Titel
- Auteur
- Uploader
- Of iemand anoniem wil zijn
- Timestamp upload
- Datum bestand gemodificeerd
- Vak*
- Leerjaar*
- niveau
- Methode > hoofstuk(ken) > paragraaf/paragrafen
- Tags (optioneel)
- Grootte (MB)
- Extensie (docx etc)
- #e revisie
- type samenvatting
